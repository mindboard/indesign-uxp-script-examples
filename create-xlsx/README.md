
# about

Creating a xlsx file from uxp InDesign scripting.


# usage

Check deno version:

```
$ deno --version
deno 1.30.3 (release, x86_64-apple-darwin)
v8 10.9.194.5
typescript 4.9.4
```

Create an idjs file:

```
$ deno bundle main.js > create-xlsx.idjs
```

and then copy the create-xlsx.idjs to your InDesign Script Folder.
